# npmPackageVerifier

## How to enter the virtual environment:
Mac:
    
    source myenv/bin/activate

Windows: 
    
    myenv\Scripts\activate

## How to run the nscan tool as a path variable 
Mac:

    chmod +x nscan
    
    ./nscan install <package name>
